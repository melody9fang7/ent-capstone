# UCI STATS 170B ENT Capstone
This is the repository for the otolaryngology data science capstone.

## DATA CLEANING
We recommend having a data folder.

### NSQIP
For NSQIP, we recommend having this folder layout:

└── ent-capstone/

    └── data/
    
        ├── nsqip_sav
        
        ├── nsqip_sav_filtered
        
        ├── nsqip_new
        
        └── nsqip
        
Keep original SAV files in **nsqip_sav**, redirect filtered sav files for each year to **nsqip_sav_filtered**, redirect the csv for each year to **nsqip_new**, redirect the combined and final filtered csv's to **nsqip**.
